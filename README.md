# 🚗 CK Leilões

Plataforma de leilões online especializada em veículos, desenvolvida com foco em desempenho, usabilidade e experiência do usuário.

## 📋 Sobre o Projeto

O CK Leilões foi criado para oferecer uma experiência moderna e intuitiva para usuários que desejam acompanhar e participar de leilões de carros pela internet.

O projeto possui um design responsivo, navegação simplificada e uma interface inspirada em grandes plataformas do mercado automotivo.

## ✨ Funcionalidades

- Página inicial moderna e responsiva
- Catálogo de veículos em destaque
- Menu mobile com botão hambúrguer
- Fechamento automático do menu ao selecionar uma opção
- Sessão de veículos disponíveis
- Área de contato
- Layout otimizado para dispositivos móveis
- Design profissional com foco em conversão

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Mobile First

## 📱 Responsividade

O projeto foi desenvolvido seguindo a metodologia Mobile First, garantindo excelente experiência em:

- Smartphones
- Tablets
- Notebooks
- Desktops

## 🎯 Objetivo

Criar uma plataforma visualmente atraente para apresentação de veículos em leilão, transmitindo confiança, credibilidade e facilidade de navegação.
